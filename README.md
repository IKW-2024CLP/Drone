# Drone

## Python Version
Python 3.6 ≥
1. Python 버전 맞추기 (conda 환경 추천)
2. `pip install -r requirements.txt` 명령 입력.
3. drone 환경 준비 완료.

## Project structure
```
/Drone
├─/dronedrone
│  │
│  ├─test
│  └─drone.py
│
└─ requirements.txt
```
